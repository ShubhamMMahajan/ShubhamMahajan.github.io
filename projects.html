Data Science Exploration

I created a program using R which would gather demographic data of all counties in California. This data was public data which could be obtained through the California website. Similiarly, I gathered data on the average household income for each county. I then compared the household income with the demographics of each county to see the demographics of the richest counties and the demographics of the poorest counties using a plot. I hope in the near future that I could perform some regressions on this data to get some more useful information.
